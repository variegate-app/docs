# Shell sort

## [<<< ---](../index.md)

Shell sort - улучшение insertion sort, которое сначала сортирует элементы на больших шагах (gap), постепенно уменьшая gap до 1.

Ключевая идея: предварительное "грубое" упорядочивание уменьшает число сдвигов на финальном проходе.

## Принцип работы

1. Выбираем начальный шаг `gap`.
2. Выполняем вставки по подпоследовательностям с этим шагом.
3. Уменьшаем `gap`.
4. Повторяем, пока `gap` не станет 1.

## Когда применять

- Средние по размеру массивы.
- Нужна простая in-place альтернатива insertion sort.
- Данные частично неупорядочены, но без строгих требований к стабильности.

## Когда лучше выбрать другой алгоритм

- Нужна предсказуемая `O(n log n)`: `./merge-sort.md` или `./heap-sort.md`.
- Нужна стабильная сортировка: `./tim-sort.md` или `./merge-sort.md`.
- Очень большие массивы общего назначения: `./quick-sort.md`.
