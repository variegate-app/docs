# Radix sort

## [<<< ---](../index.md)

Radix sort - не сравнительная сортировка, которая упорядочивает числа по разрядам (например, от младшего к старшему).

Ключевая идея: вместо прямого сравнения элементов алгоритм стабильно группирует их по цифрам каждого разряда.

## Принцип работы

1. Выбираем основание системы счисления (обычно 10 или 2^k).
2. Проходим по разрядам от младшего к старшему (LSD) или наоборот (MSD).
3. На каждом шаге делаем стабильное распределение по текущему разряду.
4. После последнего разряда получаем отсортированный массив.

## Когда применять

- Большие массивы целых чисел фиксированного диапазона/длины.
- Нужна линейная по количеству разрядов сложность.
- Можно пожертвовать дополнительной памятью.

## Когда лучше выбрать другой алгоритм

- Данные произвольные (не только целые/строки фиксированного формата): `./quick-sort.md` или `./merge-sort.md`.
- Диапазон значений маленький: `./counting-sort.md`.
- Нужна in-place сортировка с малой памятью: `./heap-sort.md`.
