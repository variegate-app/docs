# Tree sort

## [<<< ---](../index.md)

Tree sort - алгоритм, который вставляет элементы в бинарное дерево поиска, а затем получает отсортированный порядок через inorder-обход.

Ключевая идея: inorder-обход BST возвращает элементы по возрастанию.

## Принцип работы

1. По очереди вставляем элементы массива в BST.
2. Выполняем inorder-обход дерева.
3. Получаем отсортированную последовательность.
4. Переносим результат в массив.

## Когда применять

- Нужна сортировка вместе с дальнейшими операциями поиска в дереве.
- Нужны упорядоченные данные и возможны дополнительные range-операции.
- Интересен как концептуальная связь между BST и сортировкой.

## Когда лучше выбрать другой алгоритм

- Нужна предсказуемая производительность в худшем случае: `./merge-sort.md` или `./heap-sort.md`.
- Нужна лучшая средняя скорость на массивах: `./quick-sort.md`.
- Нужна стабильная сортировка: `./tim-sort.md` или `./merge-sort.md`.
