# Bucket sort

## [<<< ---](../index.md)

Bucket sort - алгоритм, который распределяет элементы по "ведрам" (bucket), сортирует каждое ведро и затем объединяет результат.

Ключевая идея: если распределение данных равномерное, то элементы быстро попадают в небольшие локальные группы.

## Принцип работы

1. Определяем число и диапазоны bucket.
2. Раскладываем элементы по bucket по их значению.
3. Сортируем каждый bucket отдельно.
4. Конкатенируем bucket в один массив.

## Когда применять

- Данные распределены более-менее равномерно.
- Известен диапазон значений.
- Нужна почти линейная сортировка в практических кейсах.

## Когда лучше выбрать другой алгоритм

- Распределение неизвестно или сильно неравномерно: `./quick-sort.md` или `./merge-sort.md`.
- Нужна стабильная универсальная сортировка: `./tim-sort.md`.
- Для целых чисел фиксированного формата чаще лучше `./radix-sort.md` или `./counting-sort.md`.
