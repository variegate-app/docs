# Counting sort

## [<<< ---](../index.md)

Counting sort - не сравнительная сортировка, которая считает количество вхождений каждого значения и восстанавливает отсортированный массив по счетчикам.

Ключевая идея: если диапазон значений `K` небольшой, можно отсортировать за `O(n + K)`.

## Принцип работы

1. Создаем массив счетчиков размера `K`.
2. Проходим по входу и увеличиваем счетчик каждого значения.
3. Превращаем счетчики в префиксные суммы (для стабильного варианта).
4. Формируем выходной массив в отсортированном порядке.

## Когда применять

- Целые числа в известном и не слишком большом диапазоне.
- Нужна очень быстрая сортировка без сравнений.
- Используется как стадия в `radix sort`.

## Когда лучше выбрать другой алгоритм

- Диапазон `K` слишком большой относительно `n`: `./quick-sort.md` или `./merge-sort.md`.
- Нужна универсальная стабильная сортировка: `./tim-sort.md`.
- Нужна сортировка in-place: `./heap-sort.md`.
