# Binary Tree

## [<<< ---](../index.md)

Binary Tree - дерево, в котором у каждого узла не более двух потомков: левый и правый.

Ключевая идея: форма дерева может быть произвольной, поэтому многие операции в общем случае не лучше линейных.

## Аналогия из жизни

Дерево решений с двумя вариантами на каждом шаге: "да/нет".

## Когда применять

- Представление деревьев выражений.
- Базовая модель для BST/heap/segment tree.
- Учебные и алгоритмические задачи.

## Когда лучше выбрать другую структуру

- Нужен гарантированный `O(log n)` поиск: `./avl-tree.md` или `./red-black-tree.md`.
- Нужен поиск по ключу в среднем `O(1)`: `./hash-table.md`.
- Нужны произвольные связи: `./graph.md`.
