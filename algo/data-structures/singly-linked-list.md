# Singly Linked List

## [<<< ---](../index.md)

Singly Linked List - связный список, где каждый узел хранит значение и ссылку только на следующий узел.

Ключевая идея: вставка/удаление около головы выполняются за `O(1)`, но доступ по индексу и поиск требуют линейного прохода.

## Аналогия из жизни

Цепочка вагонов, где каждый знает только следующий вагон.

## Когда применять

- Частые вставки/удаления в начале.
- Реализация очередей/стеков без произвольного доступа.
- Непредсказуемый рост коллекции.

## Когда лучше выбрать другую структуру

- Нужен быстрый доступ по индексу: `./array.md`.
- Нужен эффективный двунаправленный обход: `./doubly-linked-list.md`.
- Нужен быстрый поиск по ключу: `./hash-table.md`.
