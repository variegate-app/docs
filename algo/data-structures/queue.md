# Queue

## [<<< ---](../index.md)

Queue (очередь) - структура данных с политикой FIFO: первый добавленный элемент извлекается первым.

Ключевая идея: у очереди эффективны операции на концах (enqueue/dequeue), но произвольный поиск остается линейным.

## Аналогия из жизни

Очередь в кассу: кто встал раньше, тот и обслуживается раньше.

## Когда применять

- Планирование задач (job queue).
- Буферизация сообщений между сервисами/горутинами.
- BFS-обход графов и деревьев.

## Когда лучше выбрать другую структуру

- Нужен LIFO-подход: `./stack.md`.
- Нужен быстрый доступ по ключу: `./hash-table.md`.
- Нужен произвольный доступ по индексу: `./array.md`.
