# Skip List

## [<<< ---](../index.md)

Skip List - вероятностная упорядоченная структура, ускоряющая операции за счет "экспресс-уровней" ссылок.

Ключевая идея: в среднем `Search/Insert/Delete` работают за `O(log n)`, потому что переходы по верхним уровням пропускают большие блоки элементов.

## Аналогия из жизни

Многоуровневый указатель в библиотеке: на верхнем уровне переходишь между секциями, затем спускаешься к полке.

## Когда применять

- Нужна упорядоченная структура проще, чем сбалансированное дерево.
- Частые вставки/удаления вместе с поиском.
- Конкурентные сценарии (lock-free варианты skip list).

## Когда лучше выбрать другую структуру

- Нужна строгая worst-case гарантия: `./red-black-tree.md` или `./avl-tree.md`.
- Нужен только быстрый lookup по ключу: `./hash-table.md`.
- Нужна максимально простая структура: `./array.md`.
