# Red-Black Tree

## [<<< ---](../index.md)

Red-Black Tree - самобалансирующееся бинарное дерево поиска с цветовой маркировкой узлов.

Ключевая идея: ограничения по цветам и поворотам поддерживают ограниченную высоту дерева, поэтому операции стабильны по `O(log n)`.

## Аналогия из жизни

Сбалансированный шкаф папок, где при добавлении документов иногда переставляют только несколько соседних секций.

## Когда применять

- Нужна упорядоченная структура с предсказуемой производительностью.
- Реализация map/set в стандартных библиотеках и рантаймах.
- Частые вставки и удаления под нагрузкой.

## Когда лучше выбрать другую структуру

- Нужна максимально простая реализация: `./binary-search-tree.md`.
- Нужен более "строгий" баланс для чтения: `./avl-tree.md`.
- Нужен быстрый lookup без порядка: `./hash-table.md`.
