# Tree

## [<<< ---](../index.md)

Tree (дерево) - иерархическая структура данных с корнем, узлами и поддеревьями.

Ключевая идея: дерево эффективно выражает отношения "родитель-потомок", но поиск без дополнительного порядка может требовать обхода большого числа узлов.

## Аналогия из жизни

Оргструктура компании: CEO -> руководители -> команды.

## Когда применять

- Иерархии: файловые системы, меню, категории.
- Представление AST/DOM.
- Задачи обхода (DFS/BFS).

## Когда лучше выбрать другую структуру

- Нужен быстрый поиск по ключу без иерархии: `./hash-table.md`.
- Нужен гарантированный логарифмический поиск: `./avl-tree.md` или `./red-black-tree.md`.
- Нужны произвольные связи, а не иерархия: `./graph.md`.
