# Binary Search Tree

## [<<< ---](../index.md)

Binary Search Tree (BST) - бинарное дерево поиска: слева меньше, справа больше.

Ключевая идея: поиск идет только по одной ветке, поэтому средняя сложность `O(log n)`, но в вырожденном дереве может стать `O(n)`.

## Аналогия из жизни

Бумажный словарь: на каждом шаге выбираешь левую/правую часть по алфавиту.

## Когда применять

- Нужны упорядоченные данные и range-запросы.
- Нужны вставки/удаления вместе с поиском.
- Подходит, если данные относительно равномерны.

## Когда лучше выбрать другую структуру

- Нужны гарантии `O(log n)`: `./avl-tree.md` или `./red-black-tree.md`.
- Нужен lookup в среднем `O(1)` без порядка: `./hash-table.md`.
- Нужна дисковая структура: `./b-tree.md`.
