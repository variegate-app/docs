# Stack

## [<<< ---](../index.md)

Stack (стек) работает по принципу LIFO: последний добавленный элемент извлекается первым.

Ключевая идея: операции на вершине (`push/pop/top`) выполняются за `O(1)`, а поиск внутри стека - линейный.

## Аналогия из жизни

Стопка тарелок: берем и кладем только сверху.

## Когда применять

- Undo/redo в редакторах.
- Парсинг выражений и проверка скобок.
- DFS-обход и хранение контекста вызовов.

## Когда лучше выбрать другую структуру

- Нужен FIFO: `./queue.md`.
- Нужен доступ по индексу: `./array.md`.
- Нужен быстрый поиск по ключу: `./hash-table.md`.
