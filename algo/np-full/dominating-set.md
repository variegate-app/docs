# DOMINATING SET

## Формулировка

Дано: граф `G = (V, E)` и целое число `k`.

Вопрос: существует ли подмножество вершин `D ⊆ V` размера `k` такое, что **каждая вершина** из `V` либо принадлежит `D`, либо имеет соседа в `D`?

То есть `D` доминирует весь граф.

## Решение/ответ

`да`, если доминирующее множество размера `k` существует; иначе `нет`.

## Почему NP-полная (кратко)

Проверка доминирования заданного множества делается за полиномиальное время, значит задача в `NP`. `DOMINATING SET` является `NP-полной` задачей.
