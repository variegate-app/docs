# MAX-SAT (decision)

## Формулировка

Дано: булева формула в КНФ и целое число `k`.

В КНФ формула состоит из clause (дизъюнкций литералов).

Вопрос: существует ли назначение переменных, которое **удовлетворяет хотя бы `k`** clause?

## Решение/ответ

`да`, если можно удовлетворить ≥ `k` clause; иначе `нет`.

## Почему NP-полная (кратко)

Проверка назначения по числу удовлетворенных clause выполняется за полиномиальное время, значит задача в `NP`. NP-трудность следует из редукций от SAT/Max-SAT.

## Пример из реальной жизни

В антифрод-системе при доставке есть набор проверок, которые можно включить: подпись, одноразовый код (OTP) и фотофиксация.

Иногда политики конфликтуют, поэтому вместо строгого `SAT` удобнее задача `MAX-SAT (decision)`:
нужно включить проверки так, чтобы выполнить **как минимум `k` из правил**.

Переменные (включено = `true`):

- `A` — требовать подпись
- `B` — требовать OTP-код
- `C` — требовать фотофиксацию

Правила (каждое правило — это clause из 3 литералов):

1. Нужна хотя бы одна проверка: `(A ∨ B ∨ C)`
2. Если включены `A` и `B`, то нужна фотофиксация: `(¬A ∨ ¬B ∨ C)`
3. Если включены `A` и `C`, то нужен OTP: `(¬A ∨ B ∨ ¬C)`
4. Если включены `B` и `C`, то нужна подпись: `(A ∨ ¬B ∨ ¬C)`

В задаче decision задано `k` — сколько правил надо выполнить.
Для примера ниже `k = 3`, то есть нужно найти назначение `A,B,C`, которое удовлетворяет минимум 3 clauses из 4.
