# SAT (satisfiability)

## Формулировка

Дано: булева формула (произвольная, обычно в виде логического выражения) над набором переменных.

Вопрос: существует ли набор значений переменных (truth assignment), при котором формула истинна?

## Решение/ответ

`да`, если есть удовлетворяющее назначение; иначе `нет`.

## Почему NP-полная (кратко)

Проверка заданного назначения на истинность формулы выполняется за полиномиальное время, поэтому задача принадлежит `NP`. Классическая редукция показывает `NP-трудность`, значит задача `NP-полная`.

## Пример из реальной жизни

Рассмотрим кейс из логистики при обработке заказа.

- `A` — выбран способ доставки “курьер”.
- `B` — выбран способ доставки “пункт выдачи”.
- `C` — включена обязательная проверка документов.

Правила:

1. Для заказа должен быть выбран хотя бы один способ доставки: `(A OR B)`.
2. Если выбран курьер (`A`), то система обязана включить проверку документов: `(NOT A OR C)`.
3. Если выбран пункт выдачи (`B`), то система обязана включить проверку документов: `(NOT B OR C)`.
4. По условию реальной ситуации проверка документов сейчас выключена: `(NOT C)`.

Требуется понять: существует ли конфигурация `A,B,C`, которая одновременно выполняет все правила? Это ровно decision-задача `SAT`.

В этом примере ответ `UNSAT`: если выбран любой способ доставки (`A` или `B`), то правила (2) и (3) заставляют включить `C`, но по условию `C` выключен.
