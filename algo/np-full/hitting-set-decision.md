# Hitting Set (decision)

## Формулировка

Дано:

- универсальное множество `U`,
- семейство подмножеств `F = {S1, ..., Sm}` так, что `Si ⊆ U`,
- целое число `k`.

Вопрос: существует ли множество элементов `H ⊆ U` размера ≤ `k`, которое пересекает каждое множество из `F`?

То есть для каждого `Si ∈ F` выполняется `H ∩ Si ≠ ∅`.

## Решение/ответ

`да`, если существует такой `H`; иначе `нет`.

## Почему NP-полная (кратко)

Проверка того, что `H` действительно "хиттит" все множества, выполняется за полиномиальное время ⇒ задача в `NP`. Decision-вариант известен как NP-полный.
