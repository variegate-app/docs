# HAMILTONIAN PATH

## Формулировка

Дано: граф `G = (V, E)`.

Вопрос: существует ли простой путь, который проходит по **каждой** вершине ровно один раз?

Такой путь называется Hamiltonian path.

## Решение/ответ

`да`, если путь существует; иначе `нет`.

## Почему NP-полная (кратко)

Проверка заданного пути (что он простой и посещает все вершины) выполняется за полиномиальное время, значит задача в `NP`. Известно, что `Hamiltonian Path` NP-полна.
