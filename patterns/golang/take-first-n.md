# Take First N

## [<<< ---](../index.md)

Паттерн **Take First N** используется, когда из (потенциально бесконечного) потока значений нужно взять только первые `N`, а оставшиеся игнорировать и корректно завершить работу горутин‑производителей.

Чаще всего он применяется поверх других генераторов/конвейеров, чтобы ограничить объём обрабатываемых данных или время жизни конвейера.

```go
// Take возвращает канал, из которого можно прочитать не более n значений
// из входного канала valueStream. После этого выходной канал закрывается.
func Take[T any](done <-chan struct{}, valueStream <-chan T, n int) <-chan T {
	out := make(chan T)

	go func() {
		defer close(out)

		for i := 0; i < n; i++ {
			select {
			case <-done:
				return
			case v, ok := <-valueStream:
				if !ok {
					return
				}
				out <- v
			}
		}
	}()

	return out
}
```

Вызвав `Take(done, src, 5)`, мы получим новый канал, который вернёт максимум 5 значений из `src` и затем закроется, при этом при закрытии `done` чтение немедленно прекратится.
